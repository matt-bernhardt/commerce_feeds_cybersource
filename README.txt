"Import Cybersource report (e.g. payment information) using Feeds"

Features
========

- Payment/order processor for Feeds that updates order status in Drupal.

Help using Feeds
================

- The "site builder's guide to Feeds" http://drupal.org/node/622698

Installation
============

- Install & enable the Feeds (including UI), Commerce modules, and Cybersource SAWM modules
- Go to admin/structure/feeds and add a new importer
- Select the "Commerce Cybersource processor"
- In Mapping, select how raw input fields map on fields of the product entity.
